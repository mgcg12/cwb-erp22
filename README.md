<h1>deo, um ERP online</h1>

<p>ERP construído em Node.js, Express e MongoDB. Para templates, foi utilizado EJS.</p>

<p>Projeto em desenvolvimento!</p>

<h2>Instalação</h2>

1. git clone https://github.com/mgcg12/cwb-erp
2. npm install
3. node app.js
